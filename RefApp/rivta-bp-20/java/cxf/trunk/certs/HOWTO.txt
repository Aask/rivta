To create new certificates (or replace exired ones), follow this procedure:

- Delete all files in this directory, except this HOWTO.txt, cacert.pem, caprivkey.pem and gencerts.sh
- Run sh gencerts.sh
- do: svn commit -m "Certs and keystores are rectreated"
